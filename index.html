Pertanyaan Umum

Temukan jawaban atas pertanyaan yang sering diajukan

Apakah aplikasi ini gratis?

Ya! Agendaku dapat digunakan secara gratis tanpa iklan mengganggu. Kami berkomitmen untuk menyediakan aplikasi produktivitas terbaik tanpa biaya.

Apakah membutuhkan login?

Tidak. Cukup masukkan nama dan identitas opsional (NIM/Kelas) untuk pengalaman yang lebih personal. Semua data tersimpan aman di perangkat Anda.

Apakah data saya aman?

Ya. Semua data tersimpan secara otomatis dan aman di perangkatmu. Tidak ada data yang dikirim ke server kami, sehingga privasi Anda terjaga.

Bisa dijalankan di semua Android?

Minimal Android 8.0 (Oreo) ke atas dan RAM 2GB untuk performa optimal. Aplikasi ini dioptimalkan untuk berbagai ukuran layar dan dirancang responsif.
